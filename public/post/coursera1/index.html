Новые курсы на Coursera

Apr 25, 2016 · 1 minute read · Comments

Из тех курсов, которые я отобрал для себя для ознакомления:

Машинное обучение:

  • Введение в машинное обучение - совместный курс Яндекса и НИУ ВШЭ
  • Набор курсов по машинному обучению и анализу данных - опять же в партнёрстве с Яндекс Дата Фабрикой.

Математика:

  • Теория графов - Для тех кто забыл и хочет обновить знания по графам.
  • Теория вероятностей для начинающих - Тоже курс для начинающих по терверу. Кто его проспал в универе - вам сюда.
  • Комбинаторика для самых маленьких.
  • Теория случайных графов - На стыке тервера и графов.

Это всё для затравки. Так же меня порадовало, что наконец-то доделали приложения для iphone/android. IPad с собой таскать было не особо удобно.
